# SharpIR
Arduino Infra Red Sharp Lib

Based on an original work of Dr. Marcal Casas-Cartagena .

 1. Perform 25 reading of analog pin (Nb samples can be changed in .h)
 2. Sort values
 3. Convert median value to cm
